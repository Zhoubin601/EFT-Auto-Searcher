# EFT-Auto-Searcher

**EFT-Auto-Searcher** 是一款基于 Python (OpenCV) 和现代前端技术 (Vue 3 + Vite + PyWebView) 开发的自动化辅助工具。旨在通过图像识别和坐标模拟技术，提升特定场景下的操作效率。

最新版本中，我们对用户界面进行了彻底的重构，全面采用了符合 Apple Design 规范的侧边栏多标签页设计，并解耦了前后端，带来了极致的视觉体验和扩展性。

---

## 🚀 核心功能

* **🖼️ 全图搜查 (图像识别)**
  * 通过自定义区域扫描和 OpenCV 模板匹配，自动搜索预设的图标并执行点击。
  * 支持在界面上实时管理识别库（添加、删除本地截图文件）。
* **📦 弹药批量装填**
  * 专为繁琐的 7×7 网格布局设计。
  * 可通过快捷键（F1-F4）轻松采集起始弹药箱坐标、横/纵向间距以及目标弹药位置。
  * 自动双击、Ctrl+左键分装，支持自定义单个箱子的循环放置次数。
* **⚙️ 高级自定义与热键**
  * 支持一键录制启动热键。
  * 自由调节识别匹配精度 (Confidence) 和点击间隔 (Cooldown)。
  * 配置自动化持久保存 (`config.json`)。

---

## 🎨 现代化的用户界面

本项目摒弃了传统的 Tkinter，采用了全新的 **Vue 3 + Vite** 作为前端渲染引擎，结合 **PyWebView (Edge Chromium)** 呈现：
* 侧边栏分栏导航（首页、基本参数、弹药装填、图像管理）。
* 高级毛玻璃特效 (Backdrop Blur)。
* 极简且极具质感的 Apple Design 视觉规范。

## 🛠️ 安装与运行指南

### 1. 环境准备
确保您的电脑上已安装了以下环境：
- **Python 3.8+**
- **Node.js** (用于前端 Vue 项目的运行或打包)

### 2. 安装后端依赖 (Python)
在项目根目录下，安装 Python 所需的第三方库：
```bash
pip install -r requirements.txt
```
*(主要依赖包括 `opencv-python`, `mss`, `pynput`, `pywebview` 等)*

### 3. 初始化前端项目 (Node)
进入前端文件夹并安装依赖：
```bash
cd EFK-WebUI
npm install
```

### 4. 运行程序
由于采用了前后端分离架构，在开发和日常使用模式下，您需要同时开启前端服务和后端脚本：

**终端 1 (启动前端渲染服务):**
```bash
cd EFK-WebUI
npm run dev
```

**终端 2 (启动 Python 控制程序):**
```bash
# 回到项目根目录
python EFK/GameStart.py
```
> 程序启动后，将自动弹出宽 710px、高 600px 的默认窗口，并实时加载前端的 Vue 界面。
